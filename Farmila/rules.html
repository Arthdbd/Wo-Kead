Правила использования приложения и аккаунтов

За нарушение данных правил аккаунт может быть безвозвратно удалён как с уведомлением на контактную почту, так и в отдельных случаях — без предупреждения.

1. Назначение хостинга

  • Хостинг предназначен исключительно для загрузки скриншотов в качестве доказательств в отчётах.
  • Использование ресурса для иных целей строго запрещено.

2. Аккаунты

  • Каждый пользователь может иметь не более трёх (3) аккаунтов.
  • По возможности администрация будет уведомлять о нарушениях на контактную почту, указанную при регистрации.
  • За незначительные нарушения высылается предупреждение на контактную почту.
  • За серьёзные нарушения аккаунт может быть удалён без восстановления.

3. Запрещённые никнеймы

  • Запрещено создавать аккаунты с именами, никами или любыми схожими вариациями создателя проекта Farmila — Kead (Artem_Nikolskiy).
  • Запрещено использовать имена и никнеймы создателей проекта MTA Province, а также администраторов и разработчиков других серверов.
  • Запрещено использовать следующие никнеймы: Emily_Quincy, Maria_Alekseeva, Vladislav_Kiselev, Claus_Nevskiy, Platon_Seven, Denis_Manafort, Alexey_Novak, Emily_Hale и любые другие, занятые администрацией.

4. Технические ограничения

  • Запрещено засорять хостинг или предпринимать любые попытки дестабилизировать работу программы/хостинга.
  • Подобные действия ведут к безоговорочному удалению аккаунта без возможности восстановления.
